# Isola 70 – Controle de Convidados

Aplicação React + Vite para gestão de convidados, hospedagem, estoque e estatísticas do evento Isola 70.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Desenvolvimento

```sh
npm i
npm run dev
```

## Deploy

- Publicação via GitHub Pages a partir de `main/docs`.
- Scripts úteis:
  - `npm run deploy` (publica em `gh-pages`)
  - `npm run deploy:docs` (copia `dist/` para `docs/` e faz push)
